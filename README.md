# AltaRica 3.0 Open Model Library

This repository provides an open collection of **AltaRica 3.0 models** released by the **Altarica Association**.  
It gathers models used in **training sessions**, **teaching materials**, and **scientific research**, made freely available to the community for learning, experimentation, reproducibility, and further development.
